# ESPNOW2MQTT

A MQTT gateway that converts ESPNOW packets to MQTT messages and supports Home Assistant discovery.

  <img src="docs/assets/overview.png" alt="Overview" style="border-radius:8px; display:block;" />
</div>


## Usage

```bash
pnpm install
```

```bash
pnpm start
```
